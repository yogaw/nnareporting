Dampak

03

Mengapa Ini Penting Sekarang

Kompleksitas Meningkat
  • Sales, distribusi, dan collection semakin kompleks
  • Kebutuhan perubahan dari bisnis semakin cepat
Kondisi Sistem Saat Ini
  • SFA: Vendor
  • Integrasi dan Core: Vendor
  • IT internal: Sebagai penghubung

Gap yang Terjadi

  • Sistem tidak cukup fleksibel untuk kebutuhan lapangan
  • Banyak dependency antar vendor dan knowledge IT internal
Result
Sistem berjalan, namun tidak scalable
04

Lanskap Sistem Saat Ini

Current Setup
  • Multiple vendor (SFA dan Core System)
  • IT sebagai integrator
  • Data tersebar di beberapa sistem

Reality di Lapangan

  • Issue handling lambat karena dependency
  • Perubahan membutuhkan koordinasi multi pihak
  • Tidak ada centralized control
Impact
System complexity meningkat seiring waktu
